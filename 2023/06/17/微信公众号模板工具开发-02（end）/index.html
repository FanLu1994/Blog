# 引用归纳

将文章中插入的链接，以引用文献的样子放在文章最后。 这个想法来自于阮一峰的周刊。
前端由于浏览器跨域的限制，是无法根据 url 去获取到网站标题的，（不知道 wasm 是否可以）。所以需要一个后端的简单接口，使用 go 或者 python 都可以，我用的 gin 写了个接口

然后在 nuxt 中请求数据方法是 useFetch ，获取所有 url 的标题
