# iOS17 mac 测试

  1. 首先开启 remote 通道
sudo python3 -m pymobiledevice3 remote start-tunnel
  1. 然后在命令执行中，带上参数
python -m pymobiledevice3 developer dvt screenshot screen.png --rsd fdf2:3d44:d0::1 51528
